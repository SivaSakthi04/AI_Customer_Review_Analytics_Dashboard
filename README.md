# 🤖 AI Customer Review Analytics Dashboard

An AI-powered web application that analyzes customer reviews and predicts whether the sentiment is **Positive**, **Negative**, or **Neutral** using Machine Learning.

---

## 📌 Project Overview

The AI Customer Review Analytics Dashboard helps businesses understand customer feedback by automatically classifying reviews into different sentiment categories.

The project is built using **Python, Flask, SQLite, Machine Learning, Bootstrap, HTML, CSS, and JavaScript**.

---

## ✨ Features

- 🔍 Analyze customer reviews
- 🤖 Machine Learning sentiment prediction
- 📊 Dashboard with analytics
- 🥧 Pie Chart visualization
- 📈 Bar Chart visualization
- 📜 Review History
- 🔎 Search Reviews
- ✏️ Edit Reviews
- 🗑 Delete Reviews
- 📥 Export Review History (CSV)
- 📱 Responsive Bootstrap UI
